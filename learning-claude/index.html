💡 The context tax

Every conversation without custom instructions costs you a "context tax" — 2 minutes re-explaining who you are before asking your actual question. Multiply that across every Claude session in a year. Custom instructions eliminate this entirely. You set them once, and Claude knows your name, major, goals, and preferences automatically in every conversation.

Experiments
  • Write your custom instructions

    Go to Settings → User Preferences and write a brief about yourself. Loads automatically into every conversation. Takes 5 minutes. Changes everything.

    Do this in Settings, then verify

    In Claude Settings → User Preferences, paste something like: "I'm Kaitlyn, a sophomore IE student at Lehigh University. I'm into fitness and wellness. I like concise, direct answers — no filler. Use examples relevant to college or engineering." Then start a new chat and ask: "Who am I and how should you talk to me?"

    Open Settings →

    Claude knows you before you say a word. ✓

  • Create your school Project

    Projects are persistent workspaces with their own context and instructions. Create one for Lehigh coursework — you'll use it for every assignment from here.

    Try this

    Click "Projects" in sidebar → "New Project" → name it "Lehigh IE — Coursework." In Project instructions: "I'm an IE student at Lehigh. Help with coursework, research, and problem sets. Be direct, use manufacturing or operations examples."

    Open Claude →

    Your dedicated school workspace is live. ✓

  • Check and improve what Claude remembers

    As of 2026, Claude automatically builds a memory of you across conversations. View and edit exactly what it knows — then tell it what to fix.

    Try this

    Go to Settings → Capabilities → Memory. Read what Claude remembers. Then ask Claude: "What do you currently remember about me? What should I add or correct?" — it'll help you imlock it in.

    Open Settings →

    Your AI actually knows you now. ✓

  • Create a fitness & wellness Project

    Different areas of your life get different Projects — each with its own context so Claude is already briefed before you ask anything.

    Try this

    New Project: "Fitness & Wellness." Instructions: "Help with workout planning, nutrition, recovery, and wellness. I'm a college student — practical, time-efficient advice only. No generic platitudes." Then ask: "Build me a 4-week beginner strength training plan, 45 min, 3x per week."

    Open Claude →

    Two Projects, zero re-explaining. Pro setup. ✓

💡 What's a Skill?

A Skill is a saved, reusable prompt or workflow Claude can run on demand — like a shortcut for things you do repeatedly. Instead of typing the same long prompt every time, you save it as a Skill and just call it. You can use Skills others built, or have Claude build custom ones for you. Think of it as your personal AI tools library.

Experiments
  • Understand Skills by using one

    Best way to understand Skills is to experience one. Ask Claude to walk you through how they work — then try a simple one before building your own.

    Try this

    "Explain how Claude Skills work — what they are, how I use them, and show me a simple example I can run right now. Then walk me through how I'd create my own."

    Open Claude →

    Skills demystified. You know what most users don't. ✓

  • Have Claude build you a custom Skill

    This is the move that separates users from builders. Ask Claude to design and write a custom Skill for something you do repeatedly — save it, reuse it forever.

    Try this

    "Build me a reusable Skill called 'Research Paper Analyzer' for an IE student. When I paste a paper, it should: summarize key findings in plain English, identify the method, note one limitation, and suggest one follow-up question. Write the full skill prompt I can save."

    Open Claude →

    Your first custom Skill: built. That's builder mode. ✓

  • Tailor your resume to a specific job

    Generic resumes get ignored. A resume mirroring the exact language of a job description gets interviews. Claude does this in under 2 minutes.

    Try this

    Paste your resume + a job description: "Rewrite my resume to match this job description exactly. Keep everything truthful but mirror their language, lead with what's most relevant, and strengthen each bullet's impact."

    Open Claude →

    That resume just got significantly more competitive. ✓

  • Turn a class project into a portfolio piece

    Most students finish projects and move on. The ones who stand out document them as case studies — problem, approach, result. Claude writes one with you in 10 minutes.

    Try this

    "Help me turn this class project into a professional case study I can add to my portfolio. Here's what we did: [describe it]. Format: Problem → Approach → Solution → Results. Specific, quantified where possible, impressive."

    Open Claude →

    Portfolio piece done. That's a real career differentiator. ✓
